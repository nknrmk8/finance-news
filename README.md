# 金融・経済ニュース自動収集システム

信頼できる大手メディア・公的機関の**公式RSSのみ**から金融・経済ニュースを自動収集し、
スマホ対応のWebページで閲覧できるシステム。

**閲覧URL: https://nknrmk8.github.io/finance-news/**

## 仕組み

```
GitHub Actions (15分ごと)
  └─ fetch_news.py が公式RSSを収集・厳選
       └─ docs/data.json を更新してコミット
            └─ GitHub Pages (docs/) に自動反映

ページを開いた時 (ブラウザ側)
  ├─ 最新の data.json を取得(キャッシュ無効化済み)
  └─ NHK経済のRSSを直接取得してマージ(CORS対応のため即時反映できる)
```

## 収集元(厳選済み)

| 分類 | 収集元 |
|---|---|
| 報道機関 | NHK 経済 / 朝日新聞 経済 / 時事通信 / ロイター※ |
| 経済専門メディア | 東洋経済オンライン / ITmedia ビジネス |
| 海外金融 | CNBC / MarketWatch / WSJ マーケット / Financial Times |
| 編集部厳選 | Yahoo!ニュース 経済トピックス |
| 公的機関(一次情報) | 日本銀行 / 財務省 / 金融庁 |

個人ブログ・まとめサイト等は収集対象にしていない。

※ロイターは公式RSSが廃止されているため、Google News RSS 経由で
jp.reuters.com の記事のみを取得している(リンク先はロイター本体)。

## 英語記事の日本語訳

- 海外金融の英語記事は、収集時にタイトルと要約を自動で日本語訳して表示する
  (原文タイトルも併記。翻訳済み記事はキャッシュして再翻訳しない)
- 各カードの「🌐 日本語で読む」ボタンで、記事全文を Google 翻訳
  (translate.goog) 経由の日本語版で開ける

## 厳選の仕組み

- 上記ホワイトリストの公式RSSのみ収集(それ以外のソースは一切入らない)
- 総合系フィード(時事通信・東洋経済・ITmedia)は金融・経済キーワードに一致する記事のみ採用
- タイトルの重複記事は除外
- 直近72時間の記事のみ、1ソース最大15件・全体120件まで

## 手動更新

Actions タブ →「ニュース自動収集」→ Run workflow。
ローカルでは `python fetch_news.py` を実行して commit & push でも可。

## 収集元の追加・変更

`fetch_news.py` の `FEEDS` を編集する。
